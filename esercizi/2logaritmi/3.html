\[ 2\,log_4(2x - 3) - log_4(x - 1) = 1 \]
Soluzione:
L'equazione è risolta per \(\,\,\,x = \dfrac{4 + \sqrt{3}}{2}\). L'altra soluzione non è accettabile perché non rispetta le condizioni d'esistenza.

Risolvere l'equazione \[ log^2_{3}(x) - 4\,log_{3}(x) + 3 = 0 \]
Suggerimento:
Questa è una equazione logaritmica risolvibile per sostituzione. Provate a risolverla mimando il procedimento risolutivo delle equazioni esponenziali risolvibili tramite sostituzione.



Soluzione: \(\,\,\,x = 3\,\,\) oppure \(\,\,x = 27\)
