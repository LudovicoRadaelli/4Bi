Esercizio 0


Ripassare le Equazioni logaritmiche per passaggio agli argomenti disponibili QUI ed a pagg. 421, 422 del libro di testo (Volume 3A).




Esercizio 1


\[ 2\,log_4(2x - 3) - log_4(x - 1) = 1 \]

Soluzione: L'equazione è risolta per \(\,\,\,x = \dfrac{4 + \sqrt{3}}{2}\). L'altra soluzione non è accettabile perché non rispetta le condizioni d'esistenza.




Esercizio 2


\[ ln(3x^2 + 4) = ln(-x + 1) + ln(5) \]

Soluzione:
\(x = \dfrac{-5 \pm \sqrt{37}}{6}\)




Esercizio 3


Risolvere l'equazione \[ log^2_{3}(x) - 4\,log_{3}(x) + 3 = 0 \]

Suggerimento:
Questa è una equazione logaritmica risolvibile per sostituzione. Provate a risolverla? mimando il procedimento risolutivo delle equazioni esponenziali risolvibili tramite sostituzione.

Soluzione: \(\,\,\,x = 3\,\,\) oppure \(\,\,x = 27\)
