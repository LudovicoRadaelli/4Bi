Interpolazione lineare

Vogliamo costruire una funzione \(x = f(t)\) sull'intervallo \([0\,,\,\,1]\) tale che \[ \begin{cases} f(0) = 3 \\\\ f(1) = 7 \end{cases} \] Per \(t = 0\) la funzione assume il valore iniziale \(3\).
Per \(t = 1\) la funzione assume il valore finale \(7\)

Possiamo cercare \(f\) in modo che sia un polinomio di primo grado, quindi della forma \[ f(t) = at + b \quad con\,\,\,a,b\,\in\mathbb{R} \] Imponiamo le condizioni \[ \begin{cases} f(0) = 3 \\\\ f(1) = 7 \end{cases} \,\,\Longrightarrow\,\, \begin{cases} a \cdot 0 + b = 3 \\\\ a \cdot 1 + b = 7 \end{cases} \] \[ \,\,\Longrightarrow\,\, \begin{cases} b = 3 \\\\ a + 3 = 7 \end{cases} \,\,\Longrightarrow\,\, \begin{cases} b = 3 \\\\ a = 7 - 3 = 4 \end{cases} \] Dunque la funzione \(f\) che soddisfa le condizioni iniziali ha forma \[ f(t) = 4t + 3 \]


Nel file Geogebra è rappresentata la posizione del valore \(f(t)\) al variare di \(t\) tra \(0\) ed \(1\). Premete il tasto "Inizia" per avviare l'animazione. Potete modificare da tastiera il valore iniziale e finale.


Osserviamo che in generale se \[ \begin{cases} f(0) = a \\\\ f(1) = b \end{cases} \] il polinomio assume forma \[ f(t) = (b - a)t + a \]
